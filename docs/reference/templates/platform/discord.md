# Discord Platform Description

If you are running on the Discord platform, please note the following features:

## Message Types

- **Text**: Regular text messages
- **Emoji**: Server custom emoji and standard Emoji
- **Images**: Send directly or embed links
- **Embed**: Rich text message cards
- **Thread**: Replies within threads

## Message Format

### Supported Formats

- **Markdown**:
  - **Bold** `**text**`
  - *Italic* `*text*`
  - ~~Strikethrough~~ `~~text~~`
  - `Code` `` `code` ``
  - ```Code block``` ` ```code``` `
- **Links**:
  - Wrap links with `<>` to suppress embed preview: `<https://example.com>`

### Unsupported Formats

- **Don't use Markdown tables**: Use bullet point lists instead
- Complex formats may cause display issues

## Channel Types

- **Text Channel**: Regular conversation
- **Voice Channel**: Audio conversation
- **Forum/Post**: Long content discussion
- **Thread**: Topic branches
