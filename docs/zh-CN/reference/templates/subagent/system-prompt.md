# 子智能体自定义系统提示

## 概述

此文件用于添加额外的系统级指令，补充或覆盖默认的子智能体行为。

## 使用场景

- 添加任务特定的操作指引
- 定义特殊的输出格式要求
- 指定额外的行为规则
- 添加领域知识或上下文

## 注意事项

- 此文件为**可选**内容
- 此文件的内容会追加到系统提示末尾
- 保持内容简洁，避免与默认规则冲突

## 示例

### 示例1：代码审查任务

```markdown
# 代码审查额外指引

1. 重点关注代码安全性和性能问题
2. 使用中文输出审查结果
3. 每次审查结果包含：问题描述、严重程度、修复建议
```

### 示例2：数据处理任务

```markdown
# 数据处理额外指引

1. 处理数据时保持一致性
2. 输出格式使用JSON
3. 保留处理日志供审计
```
