# 🎭 TTS 语音功能

如果你有 TTS（文字转语音）工具（如 index-tts、qwen-tts 或其他本地部署的 TTS 服务），可以使用语音功能增强用户体验。

## 使用场景

### 讲故事

当用户要求讲故事、讲小说、说书时，使用语音功能：

- 使用 TTS 讲述故事情节
- 选择适合故事情节的声音
- 在群聊中让大家"听"故事

### 电影/视频摘要

将电影或视频的主要内容用语音总结：

- 提取关键信息
- 用自然的口吻叙述
- 适合通勤时收听

### "故事时间"场景

创造性使用语音：

- 诗歌朗诵
- 幽默段子
- 生日祝福
- 欢迎新成员

## 声音选择

根据内容选择合适的声音：

| 场景 | 推荐声音 |
|------|----------|
| 讲故事 | 叙事型、温和 |
| 新闻摘要 | 专业、清晰 |
| 幽默内容 | 活泼、有趣 |
| 正式场合 | 稳重、专业 |

## 使用示例

```json
{
  "tool": "tts_speak",
  "parameters": {
    "text": "今天我们要讲的故事是...",
    "voice": "narrator",
    "speed": 1.0
  }
}
```

## 最佳实践

1. **选择合适场景**：语音适合长内容、故事、摘要
2. **控制时长**：避免过长的语音消息
3. **提前告知**：发送语音前说明内容概要
4. **考虑受众**：在群聊中询问是否想听
5. **互动结合**：语音后可以继续文字交流

## 注意事项

- 在不支持语音的平台会自动转为文字
- 注意语音消息的大小限制
- 尊重用户偏好设置
